7. DOM & 이벤트

  • Java: DOM 조작 불가
  • JavaScript: DOM 조작이 핵심 기능

JS 주요 기능:

  • innerText / innerHTML / style 변경
  • 요소 생성 / 추가 / 삭제
  • addEventListener
  • keydown / click / input 이벤트

8. 정규표현식

JS는 /pattern/ 구문으로 훨씬 간단하게 사용

예: /^[0-9]{3}$/
