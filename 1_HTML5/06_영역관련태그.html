영역을 나누는 형식(성질)

block 형식

- 항상 새로운 줄에서 시작
- 이전 요소의 아래에 위치(블럭이 아래로 쌓임)
- 크기를 지정하는 width, height 속성 사용 가능

EX) p/pre(문단 나누기), h1 ~ h6(제목 영역 나누기), 
    hr (수평선 사용하여 화면 나누기), ul/ol/li (목록),
    div (영역 나누기)
  
inline 형식

- 동일한 줄에 다른 inline 요소와 나란히 배치 (블럭이 옆으로 배열)
- 크기를 지정하는 width, height 속성 사용 불가 (콘텐츠의 너비만큼 차지)

EX) Strong/b, em/i, span(한 줄을 나누는 용도)

block(div) / inline(span) 차이점 1 : 줄바꿈(개행)

div 태그

block 형식의 태그는 한 줄을 모두 차지하기 때문에 뒤에 오는 내용은 다음 줄에 출력된다

첫 번째 영역
두 번째 영역
세 번째 영역

span 태그

inline 형식 태그는 한 줄 내에서 일정한 칸을 차지하기 때문에 뒤에 오는 내용은 개행되지 않고 옆에 붙어 출력된다

첫 번째 영역 두 번째 영역 세 번째 영역

  • block : 사각형 박스 형태로 영역을 지정
  • inline : 내부에 작성된 내용(content) 단위로 영역을 지정
